## how to run the app.
1. `cd ./extension`
2. `npm run build` 
3. Navigate to [chrome://extensions/](chrome://extensions/)
4. Click the "Developer mode" toggle switch in the top right of the window
5. Click the "Load unpacked" button in top left of the window
